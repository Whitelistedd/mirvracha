### Установить и Запустит веб-сайт

1. Клонируйте репозиторий

   ```sh
   git клон https://github.com/Whitelistedd/mirvracha.git
   ```

2. Установите пакеты NPM

   ```sh
   npm install
   ```

3. Запустите веб-сайт

   ```sh
   npm run dev
   ```
